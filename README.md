# Achievement-Acceptance-model
Empirical testing of a new behavioral Economis model of happiness: the Achievement–Acceptance model.


## Overview

The Achievement-Acceptance-Model is a conceptual framework examining how individuals balance achievement and acceptance in relation to happiness. This repository contains work on testing predictions derived from the model using parametric econometric methods. It covered the full research process from problem definition to initial data handling and analysis setup.

---

## Objective

- Test predictions derived from the Achievement-Acceptance-Model.
